About SentimentPro

SentimentPro is a machine learning-powered web application for analyzing the sentiment of user reviews. It allows users to quickly determine whether feedback is positive, negative, or neutral, and provides insights using Excel file uploads and interactive charts.

Features:

  • Instant sentiment prediction of text reviews
  • Bulk analysis via Excel file upload
  • Interactive charts for visual insights
  • Modern, responsive UI
